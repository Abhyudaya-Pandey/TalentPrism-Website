# TalentPrism Website

This repo contains the static site for TalentPrism.

## New: Use Cases
We added a Use Cases tab and 5 pages inspired by Manatal content. All checklists use green checks only (no red crosses):

- `pages/use-cases.html` – Overview and quick links
- `pages/use-case-enterprise.html` – Enterprise Hiring
- `pages/use-case-high-volume.html` – High‑Volume Recruiting
- `pages/use-case-startups.html` – Startup Growth
- `pages/use-case-bpo.html` – BPO Recruitment

These pages reuse the shared components defined in `css/styles.css` (inner hero, section headings, card grids, and `.check-list`).

## Preview
Open any of the HTML files in a browser, or use a static server.
